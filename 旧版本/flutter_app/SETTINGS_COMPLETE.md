# 设置项补全说明

## 已补全的设置项

### 1. 去重检测 (Dedupe)
| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 相似度阈值 | 3 | 0-32，越小越严格 |
| 保留策略 | 最大分辨率 | 首个/最大分辨率/最大文件/最新/最旧 |
| 处理动作 | 删除重复 | 仅列出/删除重复/移动重复 |
| 移动目标路径 | 未设置 | 选择"移动重复"时指定目标文件夹 |

### 2. 格式转换 (Convert)
| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 输出格式 | PNG | JPG/PNG/WebP/ICO |
| 质量 | 85 | 1-100，仅JPG/WebP有效 |
| PNG3压缩 | 关闭 | PNG优化压缩 |
| 同格式也转换 | 关闭 | 是否重新编码相同格式 |
| ICO尺寸 | 256 | 可选16/32/48/64/128/256 |
| 非方图处理 | 中心裁切 | 中心裁切/左上裁切/等比填充/保持原样 |

### 3. 分类整理 (Classify)
| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 比例分类 | 启用 | 按宽高比例分类 |
| 自定义比例 | 16:9, 16:10, 4:3, 3:2, 5:4, 21:9, 1:1 | 可自定义 |
| 容差 | 5% | 0-20% |
| 不匹配吸附 | 启用 | 自动归类到最近比例 |
| 形状分类 | 关闭 | 按形状分类 |
| 方形文件夹名 | zfx | 可自定义 |
| 横向文件夹名 | hp | 可自定义 |
| 纵向文件夹名 | sp | 可自定义 |
| 方形容差 | 0.01 | 0.01-0.5 |

### 4. 批量重命名 (Rename)
| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 命名模板 | {name}_{index}.{fmt} | 支持{name}, {ext}, {fmt}, {index} |
| 起始序号 | 1 | 从几开始 |
| 序号步长 | 1 | 每次递增几 |
| 序号宽度 | 3 | 自动补零宽度 (0=不补) |
| 覆盖策略 | 覆盖原有 | 覆盖原有/跳过已存在/自动改名 |
| 按格式分组 | 关闭 | 每种格式独立编号 |

### 5. 一条龙模式 (Pipeline)
| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 去重开关 | 启用 | 是否执行去重步骤 |
| 转换开关 | 启用 | 是否执行转换步骤 |
| 分类开关 | 启用 | 是否执行分类步骤 |
| 重命名开关 | 启用 | 是否执行重命名步骤 |

## 待实现功能

1. **跳过格式过滤** - 允许用户选择跳过某些格式的处理
2. **删除源文件** - 处理完成后删除原始文件
3. **线程数配置** - 允许用户调整并发线程数
4. **预览模式** - 实际执行前先预览结果
5. **日志筛选** - 按阶段/关键词筛选日志
6. **批量处理进度** - 显示详细的处理进度信息

## 使用建议

1. **默认配置** 已优化为最常用的设置，适合大多数场景
2. **一条龙模式** 推荐配置：
   - 去重：阈值3，保留最大分辨率，删除重复
   - 转换：PNG格式，质量85，同格式不转换
   - 分类：启用比例分类，容差5%
   - 重命名：模板 `{name}_{index}.{fmt}`，宽度3
3. **首次使用** 建议先选择少量文件测试，确认结果符合预期
